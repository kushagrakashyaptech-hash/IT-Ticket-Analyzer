# 🖥️ IT Support Ticket Analyzer

## 📌 Project Overview

The IT Support Ticket Analyzer is a machine learning project built using Python that analyzes IT support tickets and predicts whether a ticket will be resolved within SLA (Service Level Agreement) or not.

This project helps organizations understand:

* Which departments raise the most tickets
* Peak hours for ticket creation
* Average ticket resolution time
* SLA breach prediction (Yes / No)
